I had [previously posted](http://www.cbulock.com/2005/03/camerons_thoughts_mozilla_search.html "Cameron's Thoughts Mozilla Search") about the Mozilla search plugin I created for this site. That will work in any Mozilla based browser such as Netscape or Firefox.  Now, if you check the [plugin page](http://www.cbulock.com/plugins.html "Plugins"), I also have a search plugin that works in the upcoming Internet Explorer 7.  I didn't have to actually 'create' this as Microsoft actually did something really smart and their search box supports [OpenSearch](http://opensearch.a9.com/ "OpenSearch").  As this site is already has OpenSearch 1.1 setup, I just have to create a link for IE to see the OpenSearch description document.  For more info on how to point IE to this, check out the [IE7 search provider directory](http://www.microsoft.com/windows/ie/searchguide/default_new.mspx "Add Search Providers to Internet Explorer 7").
